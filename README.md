# Hello!

This is *Temporality*, an incremental based around time and time-dilation.

Because the game doesn't contain its own information button/panel, I'll explain every mechanic below, for now.

# Codex

## Years

*Years* are the primary currency, allowing you to buy *Dilators*, generators producing *Dilation*.

At 1.797 * 10<sup>308</sup> *Years*, the game currently ends.

## Dilation

*Dilation* is a currency secondary to *Years*, producing the latter. The more *Dilation* you have, the faster *Years* will be produced/generated.

*Dilation* is produced by *Dilators*.

## Dilators

*Dilators* Are generators numbered 1 through 10. Each *Dilator* has two main values: an *amount* and a *multiplier*,
which multiply together to produce *Dilation*, or the *Dilator* one below. e.g. The 10th *Dilator* produces the 9th, the 9th the 8th, so on.

The first *Dilator* produces *Dilation*.

When reaching a certain threshold in *amount*, *Dilators* can be *compressed*, a process which *compresses* the current *multiplier* into a single *Dilator*,
making each new *Dilator* count for a heavily reduced portion of the original *multiplier*, resulting in a potentially huge gain in production.

## Compression

When a *Dilator* reaches a certain threshold in its *amount*, it can be *compressed* into one *Dilator* with a heavily reduced *multiplier*.

Each new *Dilator* produced inherit the *multiplier* gained from the previous *compression(s)*, which makes it produce much more.
